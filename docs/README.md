## Information Architecture

```text
docs/
├── architecture/      # Architecture diagrams and ADR records
├── api/               # Plugin and extension developer guides
├── manual/            # End-user guides and operational usage
└── ROADMAP.md         # Product roadmap and planning
```

> Note: On Windows file systems, ROADMAP.md and roadmap.md are treated as the same file. The canonical file name is ROADMAP.md.
